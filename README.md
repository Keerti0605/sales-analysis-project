# 📊 Sales Analysis Project

## 📌 Problem Statement

Analyze sales data to identify trends, improve profitability, and support business decision-making.

## 🛠️ Tools Used

* Python (Pandas)
* SQL (SQLite)
* Power BI

## 🔄 Project Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL Queries for business insights
4. Interactive Dashboard using Power BI

## 📊 Key Insights

* Technology category generates the highest revenue
* South region is the most profitable
* Central region shows lower performance and needs improvement
* Certain products contribute significantly to total sales
